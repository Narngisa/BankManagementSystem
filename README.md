# Bank Management System
This project is a simple banking system built as a console application in Java to practice OOP concepts.

## Language
Pure Java `OpenJDK 26.0.1`

## Feature
- Add Customer
- Create Account
- Deposit
- Withdraw
- Transfer
- Show Statement
- Show All Customers
- Show Account by Customer

## Commands Complie Java
1. `javac utils/*.java Main.java`
2. `java Main`
